## Tech stack:
**Frontend**
- **JavaScript**
- **React**
- **Styled components**
- **Vercel**

**Backend**
- **Express**
- **Passport** for authentication
- **JWT** for session control
- **Mongoose** for MongoDB communication
- **Jest** for testing
- **MongoDB**
